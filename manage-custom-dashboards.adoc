= Manage custom dashboards
:icons: font
:imagesdir: ./media/

[.lead]
Manage your custom dashboards from the dashboard catalog. From there, you can view details, refresh data, edit dashboard settings, or delete dashboards you no longer need.

.About this task

All users in your Workload Factory account can access all custom dashboards. Each dashboard shows basic details such as the creator's name, the creation date, and whether someone is currently editing it.

Custom dashboards are read-only. They cannot change your storage or workload settings. You can edit the dashboard itself, but not the resources it monitors.

.Before you begin

You must have at least one custom dashboard in your Workload Factory account.

== Access the dashboard catalog

.Steps

. Log in using one of the link:console-experiences.html[console experiences^].
. From the menu, select *Workloads Overview* and then *Custom dashboards*.
+
The dashboard catalog shows all custom dashboards in your Workload Factory account.

. Review the dashboard list.

== Refresh dashboard data

Dashboard data loads when you open a dashboard. You can refresh it to see the latest information.

The dashboard reloads data from your workloads. The last updated time changes to show when the data was refreshed.

== Edit a dashboard

You can edit a dashboard to change what it shows, including the metrics and charts.

.Steps

. In the *Custom dashboards* page, locate the dashboard you want to edit.

. From the *Actions* menu for that dashboard, select *Edit dashboard*.
+
If another user is editing the dashboard, you will see a message such as “Currently being edited by [user name].” Wait for them to finish or coordinate with them.

. In the AI dialog, describe the changes you want to make.
+
This process is similar to creating a new dashboard. You can change the workloads, metrics, or time ranges the dashboard tracks.

. Review the changes and select **Save**.

.Result

The dashboard updates with your changes. The last modified date also updates.

== Delete a dashboard

Delete dashboards you no longer need. This action is permanent and cannot be undone.

.Steps

. In the *Custom dashboards* page, locate the dashboard you want to remove.

. From the *Actions* drop-down menu for that dashboard, select *Delete*.

. Confirm the deletion.

.Result

The dashboard is permanently removed from your Workload Factory account, and all users lose access to it.

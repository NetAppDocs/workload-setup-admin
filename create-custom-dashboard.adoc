= Create a custom dashboard
:icons: font
:imagesdir: ./media/

[.lead]
Create custom dashboards by describing what you want to monitor in plain language. The AI guides you through the setup so your dashboards match your workloads and priorities.

.About this task

Custom dashboards use an AI-guided dialog to understand what you want to visualize. You describe your monitoring needs in natural language, and the system asks clarifying questions to ensure the dashboard meets your requirements. 

Once created, you can refresh the dashboard to see updated data.

Remember that custom dashboards are read-only by design and cannot make changes to your storage or workload configurations.

.Before you begin

* You need permission to access the resources you want to monitor on your dashboard.
* Set up your workloads in Workload Factory before creating dashboards to monitor them.

== Create a dashboard

.Steps

. Log in using one of the link:console-experiences.html[console experiences^].
. Select the menu image:workloads-menu-icon.png[The hamburger menu icon is used to navigate to workloads like storage, eda, ai, databases, vmware, and administration.] and then select *Workloads Overview* > *Custom dashboards*.
+
If this is your first dashboard, you'll see a blank screen with sample prompts to help you get started. You can select *Copy prompt* to copy a sample prompt into the AI dialog text box.

. Select *Create new dashboard*.
. In the AI dialog, describe what you want to monitor.
+
Describe it in plain language. The AI will ask a few questions to better understand your needs. 

. Select image:arrow.png[The menu icon is used to begin generating dashboards., width=18].

. Respond to the AI's clarifying questions.
+
The AI might ask you to choose from several options to better understand what you need. Answer each question and select *Continue*, or select *Skip* to move past optional questions.

. After answering all questions, select *Continue* to create your dashboard.

.Result

Your custom dashboard is now in the dashboard catalog. All users in your Workload Factory account can access and view it.

.What's next
After creation, you can:

* Refresh dashboard data to see the latest information  
* Edit dashboards to change what they show  
* Delete dashboards you no longer need  
* View dashboard details, including who created it and when it was last updated
